# dogfooding-page

A landing page for my dogfoofing webapp.

## Development

To set up the development environment, follow these steps:

1. **Install dependencies**:

   ```bash
   pnpm install
   ```

2. **Start the development server**:

   ```bash
   pnpm run dev
   ```

3. **Build the project**:

   ```bash
   pnpm run build
   ```

4. **Lint the code**:

   ```bash
   pnpm run lint
   ```
